公司的 Vue 專案有導入 i18n，不過 Vue 專案本身沒有 Nuxt 那麼多方便的預先配置，  
每個元件都要 import 一次 i18n 再把 t 函式拉出來，實在是有點繁瑣：

```ts
import { useI18n } from 'vue-i18n';

const { t } = useI18n({});
```

還好這個套件本身提供良心方法 `globalInjection`：
```ts
const i18n = createI18n({
  locale: 'zh',
  legacy: false,
  messages,
  globalInjection: true, // 加入這行
});
```
